Game 2048 written by c++ with SDL2

Horizontal size(Screen height): 900 pixel
Vertical size(Screen width): 627 pixel

Control:
	+Key Up, Key Down, Key Left, Key Right: Control game
	+Key F1: New game
	+Key F2: Save data
	+Key F3: Play/Pause the music

Folder "Game" including:
			+2048.exe
			+SDL2.dll
			+SDl2_image.dll
			+SDL2_mixer.dll
			+2048 Ban Chinh Thuc lan 2.bmp
			+Score.txt
